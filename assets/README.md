Branding assets
===============

Place the official Frequenz logo in this folder to brand the app.

Supported filenames (first match is used):
- `frequenz-logo.png`
- `frequenz_logo.png`
- `logo.png`
- Any `*frequenz*.png` or `*logo*.png`
